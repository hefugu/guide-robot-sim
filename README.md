# マップベース案内ロボットシミュレーション

## 目的

白黒2Dマップを用いて、建物内で利用者を目的地まで案内するロボットの経路探索・安全停止を検証する。

## 構成

- 親機モード
  - マップ読み込み
  - ロボット待機位置の設定
  - ロボット初期向きの設定
  - 障害物の設定

- 子機モード
  - 利用者位置の設定
  - 目的地の設定
  - 案内開始リクエスト

## 現在の仕様

- 白黒マップ画像を読み込む
- 白を通行可能、黒を壁として扱う
- 1マス = 25cm
- 親機でロボット待機位置を設定する
- 子機で利用者位置と目的地を設定する
- 障害物を紫色で配置できる

## 今後の実装予定

- A* または BFS による経路探索
- 待機位置 → 利用者位置 → 目的地 の2段階案内
- シミュレーション上でのロボット移動
- 障害物検知時の停止と再探索
- 経路から FORWARD / LEFT / RIGHT / STOP の命令列生成
- 既存Wi-Fiラジコンへの実機接続
